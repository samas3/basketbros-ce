# BasketBros CE

## 安装步骤（仅支持Chrome/Edge等Chromium浏览器）

1. [下载插件](https://github.com/samas3/basketbros-ce/releases/latest)
2. 打开浏览器，输入地址: `chrome://extensions/`
3. 打开"开发者模式"
4. 选择插件（.crx）并拖入浏览器窗口
5. 刷新页面

### 如果安装失败……（包无效：CRX_HEADER_INVALID）
1. 打开“显示文件扩展名”
2. 将下载的*.crx改为*.zip并解压
3. 把解压后的文件夹拖入浏览器窗口
